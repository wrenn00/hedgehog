# 🦔 고슴도치 아이스크림 가게

나만의 고슴도치 아이스크림을 조합하는 7단계 인터랙티브 웹 앱입니다.

## 데모

GitHub Pages 배포 후 링크 추가 예정

## 기능

| 단계 | 내용 |
|------|------|
| 1단계 | 콘 선택 (와플콘 / 컵 / 슈가콘 / 컵케이크) |
| 2단계 | 레이어 수 선택 (1~4층) |
| 3단계 | 층별 맛 선택 (바닐라 / 초콜릿 / 딸기 / 말차 / 블루베리 / 망고) |
| 4단계 | 고슴도치 표정 선택 (해피 / 졸려 / 새침 / 신남) |
| 5단계 | 토핑 선택 — 복수 선택 가능 (초코칩 / 스프링클 / 체리 / 웨하스 / 견과류 / 휘핑) |
| 6단계 | 아이스크림 이름 짓기 |
| 7단계 | 완성 SVG 일러스트 확인 + PNG 이미지 저장 |

## 기술 스택

- **단일 파일** `index.html` — HTML / CSS / JavaScript 올인원
- 외부 라이브러리 없음 (Google Fonts `Jua`만 CDN)
- SVG 일러스트 전부 JavaScript로 생성 (고슴도치 · 콘 · 토핑)
- 이미지 저장: SVG → Canvas → PNG (html2canvas 미사용)

## 로컬 실행

```bash
# 클론 후 index.html을 브라우저로 열면 끝
open index.html
```

또는 간단한 서버가 필요한 경우:

```bash
npx serve .
# http://localhost:3000
```
